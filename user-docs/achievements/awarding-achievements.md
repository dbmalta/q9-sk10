# Awarding Achievements

## From a member's record

1. Open the member record
2. Go to the **Achievements** tab
3. Click **Award Achievement**
4. Select the achievement type and enter the date awarded
5. Add a note if relevant
6. Save

## Bulk awarding

To award the same achievement to multiple members at once:

1. Go to **Achievements → Bulk Award**
2. Select the achievement type
3. Search for and select the members
4. Set the award date and save

## Viewing awards

A member's full achievement history is shown on the **Achievements** tab of their member record.
