# Defining Achievements

Achievement definitions must be created before they can be awarded to members.

## Creating a definition

1. Go to **Achievements → Manage Definitions**
2. Click **Add Achievement**
3. Enter:
    - **Name** — e.g. "Wood Badge"
    - **Category** — e.g. "Leadership Training"
    - **Description** (optional)
4. Save

## Editing and deactivating

Open an existing definition to edit its details. Definitions that are no longer in use can be deactivated — they will no longer appear in the award form but existing awards are retained.
