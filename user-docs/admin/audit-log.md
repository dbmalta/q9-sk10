# Audit Log

The audit log records all significant actions taken in the system — who did what and when.

## Viewing the log

Go to **Admin → Audit Log**. You can filter by:

- Date range
- User
- Action type (create, update, delete)
- Record type (member, event, etc.)

## What is logged

- Member record created, edited, or deleted
- User login and logout
- Role assignments changed
- Settings changed
- Backup created or downloaded

## Retention

Audit log entries are retained indefinitely and cannot be deleted from the UI.
