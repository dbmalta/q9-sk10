# Language Management

ScoutKeeper supports multiple languages. The interface language is selected per-user from the account menu.

## Available languages

The built-in languages shipped with ScoutKeeper are shown in **Admin → Language Management** along with their completion percentage.

## Changing the system default language

Go to **Admin → Settings** and change the **Default Language**. New users will see this language until they change their own preference.

## Editing translations

Administrators can edit translation strings directly in **Admin → Language Management**. Select a language, then search for or browse strings to update.

!!! note
    Changes made in the language editor are applied immediately and are not overwritten by system updates. Built-in translations are updated with each release.

## Adding a new language

Contributing a new translation: translations are maintained in the ScoutKeeper GitHub repository. See the [Contributing guide](https://github.com/quadninemt/scoutkeeper/blob/main/CONTRIBUTING.md) for instructions.
