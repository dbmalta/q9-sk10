# Users & Logins

User accounts are separate from member records. A person can be a member without having a login, and a login account can be created for any existing member.

## Creating a login

1. Go to **Admin → Users & Logins**
2. Click **Add User**
3. Search for an existing member record to link the account to
4. Enter an email address and temporary password
5. Assign a role
6. Save — the user can log in immediately

## Editing a user

Open the user to change their role, reset their password, or disable their account.

## Disabling a user

Disabling a user prevents login without deleting the account. The member record is not affected.

## Multi-factor authentication

Users can enable MFA from their own account settings page. Administrators cannot force MFA on other accounts at this time — this is planned for a future release.
