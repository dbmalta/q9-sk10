# Contact Directory

The contact directory lists members with their contact details, grouped by section.

## Accessing the directory

Go to **Directory → Contacts**. You can filter by section or search by name.

## Privacy

The directory only shows contact details that members have provided and that your administrator has configured as visible. Medical information and other sensitive fields are never shown in the directory.

## Exporting

If you have export permission, a **Download CSV** button is available at the top of the directory. The export respects the same field visibility rules as the on-screen view.
