# Organogram

The organogram is a visual diagram of your organisation's hierarchy.

## Viewing the organogram

Go to **Directory → Organogram** to see your full organisation tree. Each node shows the name and type.

Click on a node to see its details and the members assigned to it.

## Printing

Use your browser's print function (**Ctrl+P** / **Cmd+P**) to print the organogram. The page is formatted for print output.
