# Calendar

The calendar view shows all upcoming events in a monthly or list layout.

## Switching views

Use the view toggle (top right of the calendar) to switch between:

- **Month** — traditional grid calendar
- **List** — a chronological list of upcoming events

## Navigating

Use the arrow buttons to move between months. Click **Today** to return to the current month.

## Clicking an event

Click any event to see its full details, including location and which sections it applies to.
