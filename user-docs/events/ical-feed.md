# iCal Feed

ScoutKeeper provides an iCal feed so members can subscribe to the event calendar in their preferred calendar app (Google Calendar, Apple Calendar, Outlook, etc.).

## Getting the feed URL

The iCal feed URL is shown in **Events → Calendar → Subscribe**. Copy the URL.

## Subscribing in common apps

=== "Google Calendar"
    1. Open Google Calendar
    2. Click **+** next to **Other calendars**
    3. Choose **From URL**
    4. Paste the ScoutKeeper iCal URL and click **Add calendar**

=== "Apple Calendar"
    1. Open Calendar
    2. Go to **File → New Calendar Subscription**
    3. Paste the URL and click **Subscribe**

=== "Outlook"
    1. Open Outlook Calendar
    2. Go to **Add calendar → Subscribe from web**
    3. Paste the URL and click **Import**

!!! note
    The feed is public (no login required) so the URL should be treated as semi-private. Regenerating the URL will break existing subscriptions.
