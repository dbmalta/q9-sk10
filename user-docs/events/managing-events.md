# Managing Events

The Events module provides a calendar for planning and communicating activities.

## Creating an event

1. Go to **Events → Add Event**
2. Enter the event name, date/time, location, and description
3. Choose which sections the event is relevant to
4. Save

## Editing and deleting

Open any event from the calendar or list view to edit or delete it.

## Visibility

Events are visible to all logged-in users by default. Future versions will add per-section visibility controls.
