# First Login

## The interface

After logging in you will see the main dashboard. The layout has two parts:

- **Sidebar** (left) — navigation grouped by module
- **Topbar** (top) — search, your account menu, and dark/light mode toggle

## Recommended first steps

### 1. Set up your organisation structure

Before adding members, define your group hierarchy under **Organisation Structure**. This lets you assign members to the right section.

See [Organisation Structure](../org-structure/overview.md).

### 2. Configure roles and permissions

Review the default roles under **Admin → Roles & Permissions**. Create roles that match your organisation's staff positions and assign appropriate access.

See [Roles & Permissions](../admin/roles-permissions.md).

### 3. Add members

Once your structure is in place, start adding members under **Members → Add Member**, or import a batch via **Members → Bulk Import**.

See [Managing Members](../members/managing-members.md).

### 4. Create user logins (optional)

If other staff members will use the system, create login accounts for them under **Admin → Users & Logins**.

See [Users & Logins](../admin/users-logins.md).
