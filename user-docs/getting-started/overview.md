# Getting Started Overview

ScoutKeeper is installed on your own web hosting. There is no cloud subscription — your data stays on your server.

## What you need

- A Linux web hosting account with PHP 8.2+ and MySQL 8.0+
- The ability to upload files (FTP or cPanel file manager)
- Around 15 minutes for the initial setup

## The setup process

1. **[Check requirements](requirements.md)** — confirm your hosting meets the minimum spec
2. **[Install](installation.md)** — upload the files and create the database
3. **[Run the setup wizard](setup-wizard.md)** — configure your organisation through a browser
4. **[First login](first-login.md)** — explore the interface and add your first members

## After setup

Once your site is running, the main areas you will use are:

- **Members** — adding, editing, and searching member records
- **Organisation Structure** — setting up your group hierarchy
- **Events** — managing your calendar
- **Admin** — managing user accounts and system settings
