# Requirements

## Server requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| PHP | 8.2 | 8.3 |
| MySQL | 8.0 | 8.0+ |
| Disk space | 200 MB | 1 GB+ |
| PHP extensions | `pdo_mysql`, `mbstring`, `json`, `openssl`, `fileinfo` | — |

Most standard Linux shared hosting plans (cPanel, Plesk, DirectAdmin) meet these requirements.

!!! warning "Windows hosting"
    ScoutKeeper is tested on Linux. Windows hosting is not supported.

## PHP extensions

The setup wizard will check for required extensions and warn you if anything is missing. If you need to enable an extension, contact your hosting provider.

## Browser requirements (for administrators)

ScoutKeeper works in all modern browsers. Internet Explorer is not supported.

- Chrome 90+
- Firefox 90+
- Safari 15+
- Edge 90+
