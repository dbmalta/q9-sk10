# ScoutKeeper Documentation

Welcome to the ScoutKeeper user documentation. ScoutKeeper is a free, open-source membership management system for Scout organisations of any size.

## What is ScoutKeeper?

ScoutKeeper is a web-based platform that replaces paper records and spreadsheets with a proper membership system. It runs on standard Linux shared hosting and requires no specialist technical knowledge to operate.

## Where to start
