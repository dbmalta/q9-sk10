# Adding Members

Go to **Members → Add Member** to create a new member record.

## Required fields

- First name and last name
- Date of birth
- Section assignment (requires organisation structure to be set up first)

## Optional fields

- Contact details (email, phone, address)
- Emergency contact
- Medical information (stored encrypted)
- Any custom fields your administrator has configured

## Saving the record

Click **Save Member** to create the record. The member will appear immediately in the member list.

!!! tip
    To add many members at once, use [Bulk Import](bulk-import.md) instead.
