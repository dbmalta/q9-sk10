# Custom Fields

Custom fields let you capture additional information on member records that is specific to your organisation.

Custom fields are configured by an administrator under **Admin → Settings → Custom Fields**.

## Field types

| Type | Use for |
|------|---------|
| Text | Short free-text values |
| Textarea | Longer notes |
| Date | Date values |
| Dropdown | A fixed list of options |
| Checkbox | Yes/No values |

## Managing custom fields

1. Go to **Admin → Settings → Custom Fields**
2. Click **Add Field**
3. Enter a label, choose a type, and mark as required if needed
4. Save — the field appears on all member records immediately

Reorder fields by dragging the handle on the left. Fields can be hidden but not deleted once data has been entered against them.
