# Member Registration

ScoutKeeper can accept public registration requests, allowing families to register interest online without needing an account.

## Enabling registration

Go to **Admin → Settings** and enable **Public Registration**. You can configure:

- Which sections are open for registration
- A message shown to applicants
- Whether to require Terms & Conditions acceptance

## The registration form

Registrants fill in basic details and select a section. On submission, the entry is added to the [Waiting List](waiting-list.md) for staff to review.

## Registration link

Once enabled, the public registration URL is shown in **Admin → Settings**. Share this link on your website or social media.

!!! note
    Registration does not create a user login. It only adds an entry to the waiting list.
