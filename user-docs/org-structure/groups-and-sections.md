# Groups & Sections

## Adding a node

1. Go to **Organisation Structure**
2. Click **Add Node**
3. Enter the name, choose the type, and select the parent node
4. Save

## Node types

Node types are defined by your administrator in **Admin → Settings → Level Types**. Common examples: District, Group, Section, Unit.

## Moving a node

To move a node to a different parent, open the node and change the parent field. All child nodes move with it.

## Deleting a node

A node can only be deleted if it has no members assigned and no child nodes. Deactivate the node instead if you want to retain the history.
