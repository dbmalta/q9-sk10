# Teams

Teams are used for groups that cut across the normal hierarchy — for example, a leadership team or a committee that includes members from multiple sections.

## Creating a team

1. Go to **Organisation Structure → Teams**
2. Click **Add Team**
3. Enter a name and optionally assign a team leader
4. Save

## Adding members to a team

Open the team and use the **Add Member** button to search for and add existing members. A member can belong to multiple teams.

## Teams vs. sections

| | Section | Team |
|-|---------|------|
| Appears in hierarchy | Yes | No |
| Members assigned at registration | Yes | No |
| Used for access scoping | Yes | Yes |
